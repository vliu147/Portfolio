What is this Project?

The GrowBox is a project that started with the idea of creating a self sustainable plant care system. This device would address the needs of people who want to care for plants while being away from home for long periods of time. Some of the main goals involved were to provide adequate lighting for the plant, integrate an automatic watering system and to track real time metrics of the plant through an app or webpage.

Why did we choose this?

This project allows for users to have improved plant growth while reducing the necessary resources required to maintain them. This includes remote monitoring and controls through interconnected devices for ease of convenience. Data will also be collected at intervals chosen by the user and stored to let users find patterns throughtout the plant life cycle.

How does it work?

The project was developed using a combination of modern web technologies, including React for the front end and MySQL for the back end. It employs RESTful APIs to fetch and manage data, ensuring a seamless user experience. The app contains a dashboard for multiple devices, a statistics page for data examination and a login page for data security with unique user specifications. The data is collected through the Arduino Nano and stored in a MySQL database on the EC2 server. The box works by attaching a power supply and using sensors to track data and adding necessary components, such as a water pump, to ensure the plant survival.
